## Requirements

- PHP >= 7.4
- MySQL >= 5.7.8
- Apache >= 2.4.41
- Composer >= 2.0.9

## Installation

  git clone

### Setup

```shell
$ composer install
```

> or

```shell
$ composer update
```

## Usage

### Start Server

```shell
$ php mardira serve
```

> or

```shell
$ php mardira serve --port=<your_port>
```

### Create Controller

```shell
$ php mardira make:controller ControllerName
```

### Create Model

```shell
$ php mardira make:model ModelName
```
